My Cases

Generative AI Chatbot

Developed a Generative AI Chatbot to assist with customs regulatory documents by processing multiple PDFs. Leveraged OpenAI, LangChain, Hugging Face, and ChromaDB to provide accurate, context-aware responses. The chatbot streamlines access to complex regulatory information efficiently.

Job Offer Recommendation

Developed a job offer recommendation system by matching CV skills with job offer content. The project utilized Flask, MongoDB, and SpaCy for data processing and scikit-learn for model implementation. It also incorporated Airflow for workflow automation and Matplotlib for visualization.

Age Detection

This project focuses on age detection using deep learning models such as VGG16, ResNet, and Inception. I benchmarked these models for facial age prediction using TensorFlow, Keras, and OpenCV. The project demonstrates expertise in image classification, model evaluation, and advanced data visualization using Matplotlib and Scikit-learn.

Big Data Platform

Big Data platform capable of analyzing parliamentary data and key performance indicators (KPIs) of government engagement. This project aims to improve access and analysis of government data to help with transparency and improvement of decision-making processes.

Customer Feedback Analysis of a Bank

Designed a data warehouse to analyze customer feedback for a bank using data mining techniques. Leveraged BERT for sentiment analysis, Airflow for data pipeline automation, and PostgreSQL for data storage. Visualized insights using Power BI to enhance decision-making.

Smart Trash

Developed a Smart Trash system integrating IoT technology to optimize waste management. Utilized Arduino IDE, Blynk, and physical hardware for real-time monitoring and automation. The project enhances efficiency and sustainability in waste disposal processes.

Grades Management

Built a data warehouse for grades management, centralizing transaction data through ETL processes for analysis and reporting. Used Talend for data integration, MySQL for storage, and Power BI for insightful visualizations.

Tourist Arrivals Analytics in Morocco (2012-2020)

Conducted an analysis of tourist arrivals in Morocco (2012–2020), focusing on trends by nationality at border posts using Factorial Correspondence Analysis (FCA). Leveraged Talend, MySQL, and Power BI for data management and visualization, alongside R and Excel for advanced statistical analysis.
